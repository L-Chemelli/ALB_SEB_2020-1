## Como montar o código:
1) nasm <nome_do_arquivo>.asm ou nasm16 -f obj -o <nome_do_arquivo>.obj -l <nome_do_arquivo>.lst <nome_do_arquivo>.asm
2) freelink <nome_do_arquivo>
3) <nome_do_arquivo> (rodando o programa)
